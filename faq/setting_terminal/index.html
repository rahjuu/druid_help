How to change setting for a terminal device?

You can change setting for a terminal device at device list page or the device information page.

Changing setting at device list page

  1. On the main menu, click Device > Terminal.

  2. Locate the device and click Device setting.

  3. Change the settings as needed.

    For the detailed description on the terminologies on device setting page, see the terminology section.

  4. Click Confirm.

**Changing setting at device information page** 1. On the main menu, click **Device** > **Terminal**.

  1. Click device S/N to go to the device information page.

  2. Click Device setting.

  3. Change the settings as needed.

  4. Click Confirm.
